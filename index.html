Concretify - The Analysing Tool

This project, Concretify, is an analyzing tool designed for civil engineers. It allows users to calculate compressive strength, flexural test, and tensile test for concrete structures. Built using React for the frontend and Spring Boot for the backend, it provides a user-friendly interface for engineers to input data, perform calculations, and store results in a MySQL database.
